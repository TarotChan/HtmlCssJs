显示属性

display

初始值：视元素而定

值：

  • 一般元素：none, inline, block, inline-block, run-in
  • 表元素：table, inine-table, table-row, table-row-group, table-header-group, table-footer-group, table-column, table-column-group, table-cell, table-caption
  • 表样式：list-item
  • inherit

继承性：无

详细描述：

  • 应用于所有元素
  • none：元素从文档流中删除
  • inline：行内元素，
  • block：块元素，
  • inline-block：作为一个块级元素被格式化成行内元素，像图像一样放在文本行中
  • run-in：使一个块级元素嵌入到下一个块级元素中
  • 表元素的 display 值主要应用于 XML 中，因为 HTML 样式表已经对表格式进行了初始化，例如：
    • table {display: table;}
    • tr {display: table-row;}
    • thead {display: table-header-group;}
    • tbody {display: table-row-group;}
    • tfoot {display: table-foot-group;}
    • col {display: table-column;}
    • colgroup {display: table-column-group;}
    • tr, th {display: table-cell;}
    • caption {display: table-caption;}
    • inline-table：与 inline-block 相似，将表格作为一个行内元素嵌入其他元素（本质仍然是块级元素，只是表现方式为行内元素）
  • list-item：display 为 list-item 的元素可以设置表样式 list-style

list-style

初始值：相对于各属性

值：

  • list-style-type
  • list-style-image
  • list-style-position
  • inherit

继承性：有

详细描述：

  • 应用于display: list-item的元素
  • list-style-type

    • 值：

      • circle：空心圆
      • squre：方块

      • upper-alpha：A, B, C...
      • upper-latin：
      • lower-alpha：a, b, c...
      • lower-latib：
      • upper-roman：I, II, III...
      • lower-roman：i, ii, iii...
      • lower-greek：小写希腊符号
      • armenian：亚美尼亚符号
      • georgian：乔治符号
      • none：不使用标志
      • inherit
    • 继承性：有
    • 详细描述：
  • list-style-image
    • 初始值：none
    • 值：
      • url
      • none
      • inherit
    • 继承性：有
    • 详细描述：一般在使用 list-style-image 时，还可以定义一个预备的 list-style-type
  • list-style-position
    • 初始值：outside
    • 值：
      • inside
      • outside
      • inherit
    • 继承性：有
    • 详细描述：定义图标在列表内容之外还是列表内容之内

position

初始值：static

值：

  • static
  • relative
  • absolute
  • fixed
  • inherit

继承性：无

详细描述：

  • 应用于所有元素
  • static：元素框正常生成，块级元素生成一个矩形框作为文档流的一部分，行内元素生成一个或多个行框放置在其父元素中
  • relative：元素框仍然在文档流中，相对于原位置进行位置偏移（使用top, right, bottom, left）
  • absolute：元素框从文档流中删除，相对于其包含块定位，定位后生成一个块级框
  • fixed：类似于 absolute ，不过其包含框是视窗
  • 根元素：一般为 html ，有少部分浏览器为 body
  • 包含块：对于非根元素，如果其 position 值是 static 或者 relative ，则其包含块是最近的块级框；如果其 position 值为 absolute ，则其包含块为最近的 position 值不是 static 的元素

top, right, bottom, left

初始值：auto

值：

  • [length]
  • [percentage]
  • auto
  • inherit

继承性：无

详细描述：

  • 应用于定位元素，即position值不是static的元素
  • 百分数：top和bottom对应于包含块的高度，left和right对应于包含块的宽度
  • 上述定位框已经隐含了 width 和 height 属性

值：

  • [length]
  • [percentage]
  • inherit

继承性：无

详细描述：

  • 用于除了非替换行内元素和表元素外的所有元素

  • 用于绝对定位元素

初始值：auto

  • 代表顺序的数字：1, 2, 3...
  • auto
  • inherit

继承性：无

详细描述：

  • 用于定位元素
  • 显示的优先级

float

初始值：none

值：

  • left
  • right
  • none
  • inherit

继承性：无

详细描述：

  • 用于所有元素

  • 首先，浮动会从正常的文档流删除

值：

  • auto
  • inherit

继承性：无

详细描述：

  • 用于块级元素和替换元素
  • 百分数相对于包含块的 width

height

初始值：auto

值：

  • [length]
  • [percentage]
  • auto
  • inherit

继承性：无

详细描述：

  • 用于块级元素和替换元素
  • 百分数相对于包含块的 height

padding

初始值：未定义

值：

  • [length]
  • [percentage]
  • inherit

继承性：无

详细描述：

  • 应用于所有元素
  • 在元素上放边框时，同时增加一点padding
  • padding的百分数参数基于其包含块的width计算，所以如果其父元素width改变，使用的百分数padding也会改变
  • 可以更改元素的单边内边距：padding-top, padding-right, padding-bottom, padding-left
  • 对于行内元素，默认的 padding-left 和 padding-right 会显示， padding-top 和 padding-bottom 不会显示，但当为其填充颜色时会覆盖原有区域

border

初始值：根据单个属性

值：

  • border-width
  • border-style
  • border-color
  • inherit

继承性：无

详细描述：

  • 应用于所有元素
  • 也可以对单个边框进行样式设计（border-top, border-right, border-bottom, border-left），设置格式参照border
  • 只要设置了边框，边框的样式border-style是必要的，其他都可以是默认值
  • border-width
    • 初始值：未定义
    • 值：
      • thin: 细
      • medium: 适中
      • thick: 粗
      • [length]: 长度
      • inherit:
    • 继承性：无
    • 详细描述：
      • 可自定义单边框的宽度（border-top-width, border-right-width, border-bottom-width, border-left-width）
      • 定义了边框长度以后一定要定义边框样式border-style
  • border-style
    • 初始值： none
    • 值：
      • none: 无
      • hidden: 隐藏
      • dotted: 密集点
      • dashed: 长点
      • solid: 实线
      • double: 双实线
      • groove:
      • ridge:
      • inset: 内凹样式
      • outset: 外凸样式
      • inherit:
    • 继承性： 无
    • 详细描述
      • 可自定义多种样式（border-style: top, right, bottom, left;）
      • 可自定义单边样式（border-top-style, border-right-style, border-bottom-style, border-left-style）
  • border-color
    • 初始值：元素的前景色
    • 值：
      • [color]:
      • transparent:
      • inherit:
    • 继承性：无
    • 详细描述：
      • 颜色可以是命名颜色，也可以是十六进制和RGB颜色
      • transparent用于创建一个有宽度的透明框
      • 可自定义单边框颜色（border-top-color, border-right-color, border-bottom-color, border-left-color）

margin

初始值：未定义

值：

  • [length]
  • [percentage]
  • auto
  • inherit

继承性：未定义

详细描述：

  • 应用于所有元素
  • 百分数值是相对于父元素的 width 计算的
  • 可设置单边外边距（margin-top, margin-right, margin-bottom, margin-left）
  • 可以有负外边距
  • 外边距会合并
  • 外边距是透明的，背景只会衍生到内边距
  • 左右外边距对行内元素是有影响的

background

初始值：根据各个属性

值：

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • inherit

继承性：无

详细描述

  • background没有必不可少的值，至少出现至少一个值就够了
  • background-color
    • 初始值： transparent
    • 值： [color], transparent, inherit
    • 继承性： 无
    • 详细描述：填充背景颜色或者让它透明（默认）。
  • background-image
    • 初始值：none
    • 值：url, none, inherit
    • 继承性：无
    • 详细描述：只要有背景图像就应该同时包含一个背景颜色。
  • background-repeat
    • 初始值：repeat
    • 值：repeat, repeat-x, repeat-y, no-repeat, inherit
    • 继承性：无
    • 详细描述：repeat在水平和垂直方向都平铺，repeat-x和repeat-y分别只在水平和垂直方向上平铺，no-repeat不允许平铺
  • background-attachment
    • 初始值：scroll
    • 值：scroll, fixed, inherit
    • 继承性：无
    • 详细描述：
      • scroll: 图片相对于文档定位
      • fixed: 图片相对于窗口定位
  • background-position
    • 初始值：0% 0%
    • 值：（有两个参数）
      • 第一个参数：[length], [percentage], left, center, right
      • 第二个参数：[length], [percentage], top, center, bottom
      • inherit
    • 继承性：无
    • 详细描述：
      • 放置图像的上下文是包含块的内边距
      • 关键字通常成对出现，第一个关键字表示水平方向，第二个关键字表示竖直方向
      • 如果只出现一个关键字，另一个关键字为center
      • 可以使用负长度参数是背景图像部分显示

文本属性

color

初始值：用户代理特定

值：

  • [color]
  • inherit

继承性：有

详细描述：

  • 用于所有元素
  • color可以设置元素的前景色，可以让子元素继承从而影响到边框，表单元素等
  • css颜色小试

font

初始值：根据单个属性

值：

  • font-style, font-variant, font-weight, font-size/line-height, font-family
  • 系统字体：caption, icon, menu, message-box, small-caption, status-bar
  • inherit

继承性：有

详细描述：

  • 应用于所有元素
  • font 声明中必须要有 font-size , font-family 这两个值，否则整个声明无效
  • 可以用 font 设置行高：在 font-size 之后可以设置行高，之间用一个斜杠（/）分隔
  • 可以直接使用系统字体，然后再对单个字体属性进行修改： button {font: caption; font-size: 1em;}
  • font-style
    • 初始值： normal
    • 值： italic, oblique, normal, inherit
    • 继承性： 有
    • 详细描述：normal是竖直的文本，italic和oblique都是竖直文本的倾斜版（italic对每个字都进行了一小点改进，而oblique直接把竖直文本倾斜化）
  • font-variant
    • 初始值： normal
    • 值： small-caps, normal, inherit
    • 继承性： 有
    • 详细描述：small-caps是一种小型大写字母文本，normal描述的是正常文本
  • font-weight
    • 初始值： normal
    • 值：
      • normal, bold, bolder, lighter
      • 100（最细）-900（最粗）
      • inherit
    • 继承性： 有
    • 详细描述：
      • 一般情况下nomal为400或者500，bold比normal要粗
      • lighter让字体更细
  • font-size
    • 初始值： medium
    • 值：
      • xx-small, x-small, small, medium, large, x-large, xx-large
      • smaller, large
      • [length], [percentage]
      • inherit
    • 继承性：有
    • 详细描述：
      • font-size的作用主要是给em框提供一个大小
      • 绝对大小之间的缩放因子是固定的
      • 相对大小就是把字体大小在一个梯度上向大或者向小移动
      • 百分数值是根据由父元素继承的字体大小来计算的，可以使用任何长度值来设置font-size
  • font-family
    • 初始值：用户代理指定的值
    • 值：[family-name]（字体系列）
    • 继承性：有
    • 详细描述：已定义的五种通用字体系列：serif, sans-serif, monospace, cursive, fantasy

值：

  • [length], [percentage], [number], normal, inherit

继承性：有

详细描述：

  • 应用于所有元素

vertical-align

初始值：baseline

值：

  • baseline, sub, super, top, text-top, middle, bottom, text-bottom
  • [percentage], [length], inherit

继承性：无

详细描述：

  • 应用于行内元素和表单表格
  • 基线对齐（baseline）：
  • 上标和下标对齐（super/sub）：其基线相对于其父元素的基线升高或降低
  • 顶端和底端对齐（top/bottom）：其基线顶端/底端与父元素的基线的顶端/底端对齐
  • 百分数参数根据其父元素的line-height计算，正（负）百分比使其基线相对于父元素基线升高（下降）
  • 所有的垂直对齐都会影响行高
