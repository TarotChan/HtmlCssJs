Jquery 中的 DOM

  • 查找节点
    • 查找节点文本内容：Text()返回节点的文本内容
    • 查找节点属性：attr(val)返回属性val的值，当然两个参数时将可以设置属性值
  • 创建节点
    • 可以使用工厂函数 $(val) 新建一个 jQ 对象
    • 然后使用各种插入函数进行插入
  • 插入节点
    • append() - appendTo()
    • prepend() - prependTo()
    • after() - insertAfter()
    • before() - insertBefore()
    • 后面的函数可以很方便的使用链式调用。例如 $("<li>建文档节点</li>").appendTo("div.main>ul:eq(1)");
    • 然鹅我还是搞不清楚 append() after() - prepend() before() 的区别...
    • append() prepend() 每次都只往最前或者最后追加
    • after() before() 每次都是在固定的元素前后追加
    • 插入节点也可以用来移动文档中的元素，改变其原有顺序
  • 删除节点
    • remove()
    • detach()
    • empty()
  • 复制节点
    • clone()
  • 替换节点
    • replaceWith() - replaceAll()
  • 包裹节点
    • wrap()
    • wrapAll()
    • wrapInner()
  • 遍历节点
    • children()
    • next()
    • prev()
    • siblings()
    • closest()
  • 属性操作
    • attr() - removeAttr()
  • 样式操作
    • 使用 attr() 修改 class 属性
    • addClass()
    • removeClass()
    • toggleClass()
    • hasClass()
  • HTML-DOM
    • html()
    • text()
    • val()
  • CSS-DOM
    • css() - 与 attr() 类似
    • height() - width()
    • offset()
    • position()
    • scrollTop() - scrollLeft()
  • 执行函数后绑定事件会一同消失的函数
